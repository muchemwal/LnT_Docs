# LANDTTS Digital Patient Transport Platform

import { Callout, Cards, Card } from 'nextra/components'

<Callout type="info">
  Version 2.0 — Technical and Operational Documentation
</Callout>

The LANDTTS platform is a purpose-built clinical transport management system for NHS-contracted
patient transfer services. It replaces paper-based Patient Transfer Forms (PTFs), manual dispatch,
and fragmented admin processes with a unified digital workflow from booking to financial reconciliation.

## Architecture at a Glance

| Layer | Technology |
|---|---|
| Web App | React (Vite) |
| Mobile App | React Native (Expo SDK 50) |
| API Gateway | Port 8000, routes to 19 microservices |
| Services | Express.js, ports 3001–3018 |
| Database | PostgreSQL 15+ with Prisma ORM |
| Cache / PubSub | Redis 7+ |
| Identity | Keycloak OIDC with TOTP MFA |
| Storage | Amazon S3 (signatures, PDFs, documents) |
| Integrations | Xero (accounting), Factorial HR (payroll), Mapbox (geocoding) |
