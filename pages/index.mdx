L&T Patient Transport is a full-stack platform for managing non-emergency patient transfers
across the UK. It includes a web dashboard, mobile crew app, client portal, and 18 microservices.

## Architecture at a Glance

| Layer | Technology |
|---|---|
| Web App | React 18, TypeScript, Vite, TanStack Query |
| Mobile App | React Native, Expo SDK 50, TypeScript |
| API Gateway | Kong 3.7 (DB-less declarative), port 8000 |
| Reverse Proxy | Caddy 2 (auto-HTTPS, WebSocket routing) |
| Services | 18 Express.js microservices, ports 3001–3018 |
| Database | PostgreSQL 16 (primary + audit) with Prisma ORM |
| Cache / PubSub | Redis 7 (cache, pub/sub, streams) |
| Object Storage | MinIO (S3-compatible) — signatures, PDFs, documents |
| Identity | Keycloak 24 OIDC with PKCE + TOTP MFA |
| Email | Resend API (transactional emails) |
| Integrations | Xero · Factorial HR · Deel · Mapbox · Twilio · APNs/FCM |
| Monorepo | Turborepo, npm workspaces |

## Platform Summary

- **Web Dashboard** — Dispatch board, rota management, analytics, billing, governance
- **Mobile App** — Crew workflow (job acceptance, GPS tracking, shift check-in, PTFs, medication recording)
- **Client Portal** — Booking requests, live transfer tracking
- **Backend** — 18 microservices, Keycloak SSO, PostgreSQL, Redis, MinIO
